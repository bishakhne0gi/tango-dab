#Installation & setup

1. Clone the repository
  
    ```bash
     git clone https://github.com/bishakhne0gi/tango-dab.git
     ```
2. Change the directory
    ```bash
    cd tango-dab
    ```

2. Install the node modules

    ```bash
    npm install
    ```
3. Run the server

    ```bash
    npm start
    ```
